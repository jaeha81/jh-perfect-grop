## 오프닝 (30초)

"안녕하세요. 인테리어 현장 경력 18년, AI 풀스택 빌더 이재하입니다.

오늘 발표할 JH EstimateAI는 제가 18년간 현장에서 겪은 문제를 직접 코드로 푼 시스템입니다.

한 마디로 정의하면 — 교수님 AI가 아닌, 현장의 AI입니다."

## 문제 정의 (30초)

"견적은 왜 아직도 수작업일까요?

기존 방식의 문제는 세 가지입니다.

첫째, 단가가 하드코딩되어 있어 지역·시기 변동을 반영 못 합니다.
둘째, 범용 AI는 욕실 방수 누락처럼 현장 경험 없이는 못 잡는 이상치를 통과시킵니다.
셋째, 숙련 견적사도 30분~3시간이 걸립니다.

JH EstimateAI는 이 세 가지를 동시에 해결합니다."

## 솔루션 데모 (2분 30초)

### 데모 시작

"지금 바로 데모를 보여드리겠습니다.
30평 아파트 전체 인테리어 견적을 내보겠습니다."

*[데모 버튼 클릭 → 자동 입력]*

"공사 유형 인테리어, 면적 99m2, 욕실 2개 교체 + 강마루 + 실크도배.
견적 산출하기를 누르겠습니다."

### 에이전트 파이프라인 (실행 중)

"화면 위에 5개 에이전트가 순서대로 실행됩니다.

SCANNER가 공간을 분석하고,
ESTIMATOR가 96개 공종 키워드로 수량을 산출합니다.
PRICER가 실제 단가 DB에서 금액을 계산하고,
VALIDATOR가 18년 현장 룰셋 37개로 이상치를 검증합니다.
마지막으로 REPORTER가 PDF 견적서를 생성합니다."

### 결과 확인

"결과가 나왔습니다. 약 12초입니다.

공종별 내역 14개가 자동 분류되었고,
VALIDATOR는 정상 판정 — 전문가 총평도 자동 생성되었습니다.

PDF 다운로드를 누르면 즉시 견적서가 생성됩니다."

### 비교 견적 시연

"추가 기능입니다. 저가/표준/고급 비교 견적 버튼을 누르면
동일 공간에 대해 세 가지 등급 견적을 한 번에 비교할 수 있습니다.

저가 80%, 표준 100%, 고급 125% 단가입니다.
업체마다 견적이 다른 이유를 이 화면 하나로 설명할 수 있습니다."

## 핵심 차별화 (1분)

"가장 중요한 차별화 포인트는 VALIDATOR입니다.

GPT나 Gemini에게 전문가처럼 검증해달라고 프롬프트를 쓰는 게 아닙니다.
제 18년 경험이 Python 코드 자체에 내재되어 있습니다.

욕실 방수 공사가 없으면 즉시 에러,
전기 공사 비중이 3% 미만이면 누락 의심 —
이런 룰 37개가 코드로 구현되어 있습니다.

AI가 틀려도 룰이 잡고, 룰이 놓쳐도 AI가 보완합니다."

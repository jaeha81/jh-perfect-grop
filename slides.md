## Slide 4 — 핵심 차별화: VALIDATOR

### 18년 현장 룰셋 내재화

```python
# 실제 VALIDATOR 검증 기준
- m² 당 150,000원 미만  → "인건비 미포함 수준" (error)
- 전기 비중 3% 미만     → "전기공사 누락 의심" (warning)
- 욕실 공사 150만원 미만 → "품질 위험" (error)
- 철거 없는 리모델링    → "철거비 누락" (warning)
```

**GPT/Gemini와의 차이**: 단순 계산이 아닌 현장 경험 기반 품질 검증

## Slide 6 — 데모 결과 (30평 아파트)

### 입력
- 공사 유형: 인테리어
- 면적: 99m²
- 설명: 전체 인테리어, 욕실 2개 교체, 강마루, 실크도배

### 출력 (약 12초)
- 견적 범위: **2,200만원 ~ 3,800만원**
- 기준 단가: 275,000원/m²
- 공종 내역: 14개 공종 자동 분류
- VALIDATOR: 정상 판정 + 전문가 총평 자동 생성
- PDF: 즉시 다운로드
